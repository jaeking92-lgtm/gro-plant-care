# GRO · Plant Care MVP

GRO is a mobile plant-care MVP prototype focused on helping beginners start a simple daily care routine for their first plant.

## Main Flow

Login / Guest Mode → Intro → Window Direction → Place → Pet Status → Watering Style → Plant Recommendation → Plant Registration → Home → Today Care → Growth Record

## Key Features

- Environment-based plant recommendation flow
- First-run onboarding and registration state
- 24-hour daily care clock for water, sunlight, and record routines
- Empty-state growth record screen designed for first-time users
- Bottom navigation optimized for MVP: Home / Plant / Today Care / Records
- LocalStorage-based prototype state

## Tech

- HTML
- CSS
- JavaScript
- Static assets

## Notes

This prototype is designed as a portfolio-ready UX/UI MVP. It can be deployed directly through GitHub Pages or Vercel as a static site.
